# C# Master Roadmap

> **C#** is a modern, type-safe language on **.NET**. This track covers what you need day-to-day as a backend engineer at **basic → intermediate** level—no advanced runtime internals yet.

---

## What You'll Learn

| Area | Outcome |
|------|---------|
| Language fundamentals | Types, control flow, methods, null safety |
| OOP | Classes, properties, inheritance, interfaces |
| Collections & LINQ | Lists, dictionaries, querying data in memory |
| Ecosystem | Exceptions, project structure, `dotnet` CLI |

---

## Learning Path

### 1. Language Fundamentals

| Note | Focus |
|------|--------|
