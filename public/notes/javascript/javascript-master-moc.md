# JavaScript & TypeScript Roadmap

> **JavaScript** runs the browser and Node.js backends. **TypeScript** adds static types on top—this track matches a **basic → medium** skill level: solid fundamentals, not engine internals.

---

## How the tracks fit together

| Track | Language | Goal |
|-------|----------|------|
| 1. JavaScript Fundamentals | JS | Syntax, objects, control flow |
| 2. Asynchronous JavaScript | JS | Promises, async/await, event loop intro |
| 3. TypeScript Fundamentals | TS | Types, interfaces, generics, config |

---
