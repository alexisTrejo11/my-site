# Python Overview

> Python: readable, versatile, and powerful

## What is Python?

**Python** is a high-level, interpreted programming language known for its readability and versatility. It supports multiple paradigms: procedural, OOP, and functional.

## History

| Year | Event |
|------|-------|
| 1991 | Python created by Guido van Rossum |
| 2000 | Python 2.0 released |
| 2008 | Python 3.0 released (breaking changes) |
| 2020 | Python 2 discontinued |
| 2024 | Python 3.12+ (latest) |

## Key Features

| Feature | Description |
|---------|-------------|
| Interpreted | No compilation needed |
| Dynamic | No type declarations |
| Duck Typing | If it quacks like a duck... |
| Batteries Included | Rich standard library |
| Multi-paradigm | OOP, functional, procedural |

## Python vs Other Languages

| Aspect | Python | Java | JavaScript |
|--------|--------|------|----------|
| Typing | Dynamic | Static | Dynamic |
| Compiled | No (interpreted) | Yes | Yes (JIT) |
| OOP | ✅ | ✅ | ✅ |
| Functional | ✅ | Limited | ✅ |
| Type Hints | Optional | Required | Optional |
