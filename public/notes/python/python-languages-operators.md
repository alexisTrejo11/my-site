# Operators

> Arithmetic, comparison, and logical

## Arithmetic

```python
# Basic
2 + 3     # 5 Addition
5 - 2      # 3 Subtraction
4 * 3      # 12 Multiplication
10 / 2      # 5.0 Division
10 // 3     # 3 Floor division
10 % 3      # 1 Modulus
2 ** 4      # 16 Exponent
```

## Comparison

```python
# Returns True/False
5 == 5       # Equal
5 != 3       # Not equal
5 > 3        # Greater
3 < 5        # Less
5 >= 3       # Greater or equal
5 <= 5       # Less or equal
```

## Logical

```python
True and False  # False
True or False   # True
not True      # False
```

## Bitwise

```python
5 & 3       # 1 (AND)
5 | 3       # 7 (OR)
5 ^ 3       # 6 (XOR)
~5          # -6 (NOT)
4 << 1      # 8 (Left shift)
8 >> 1      # 4 (Right shift)
```

## Assignment

```python
x = 5       # Basic
x += 3      # x = x + 3
x -= 2      # x = x - 2
```
