# Variables and Constants

> Variable assignment and naming

## Variables

```python
x = 10           # Integer
name = "Python"    # String
is_active = True   # Boolean

# Multiple assignment
a, b, c = 1, 2, 3
```

## Constants (by convention)

```python
MAX_CONNECTIONS = 100  # UPPER_SNAKE_CASE
PI = 3.14159
```

## Variable Naming

- Start with letter or underscore
- Case sensitive
- No reserved keywords

```python
_valid = 1
_private = 2
__dunder = 3
```

## Type Hints (Python 3.5+)

```python
name: str = "Python"
age: int = 30
items: list = []
```
