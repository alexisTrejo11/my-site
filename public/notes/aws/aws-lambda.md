 Created to run code without provisioning or managing servers. You pay only for the compute time that you consume—there's no charge when your code isn't running. You can run code for virtually any type of application or [backend](/learning/backend) service—all with zero administration. Just upload your code and Lambda takes care of everything required to run and scale your code with high availability.
