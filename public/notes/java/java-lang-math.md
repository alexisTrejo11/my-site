# java.lang.Math

> Math utilities

## Basic Math

```java
Math.abs(-5);      // 5
Math.max(3, 5);    // 5
Math.min(3, 5);    // 3
Math.floor(5.9);    // 5.0
Math.ceil(5.1);     // 6.0
Math.round(5.5);    // 6
Math.sqrt(16);      // 4.0
Math.pow(2, 3);    // 8.0
Math.exp(1);       // 2.718...
Math.log(Math.E);   // 1.0
```

## Trigonometric

```java
Math.sin(x);
Math.cos(x);
Math.tan(x);
Math.toDegrees(x);
Math.toRadians(x);
```

## Random

```java
Math.random();  // 0.0 to 1.0
```
