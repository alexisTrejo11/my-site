# Data Types

> Basic data types in Python

## Numeric Types

```python
# Integer
age = 30
binary = 0b1010      # 10
hex_num = 0xFF        # 255

# Float
price = 19.99
scientific = 1.5e-3   # 0.0015

# Complex
complex_num = 3 + 4j
```

## Sequence Types

```python
# String
name = "Python"
multi_line = """Multi
line"""

# List
numbers = [1, 2, 3]

# Tuple
coordinates = (10, 20)
```

## Mapping

```python
# Dictionary
person = {"name": "John", "age": 30}
```

## Set Types

```python
# Set
unique_nums = {1, 2, 3}

# Frozenset
frozen = frozenset([1, 2, 3])
```

## Boolean

```python
is_active = True
is_empty = False
```
