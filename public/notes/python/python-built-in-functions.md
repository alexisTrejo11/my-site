# Built-in Functions

> Commonly used built-in functions

## Type Conversion

```python
int()      # To integer
float()    # To float
str()      # To string
bool()     # To boolean
list()     # To list
dict()     # To dictionary
tuple()    # To tuple
set()      # To set
```

## Iterables

```python
len()      # Length
min()      # Minimum
max()      # Maximum
sum()      # Sum
sorted()   # Sorted list
reversed() # Reverse iterator
enumerate() # Index and value
zip()       # Pair iterables
all()      # All true
any()      # Any true
filter()   # Filter items
map()      # Map function
range()    # Range of numbers
iter()     # Iterator
next()     # Next item
```

## Input/Output

```python
print()    # Output
input()    # Input
open()     # File open
```

## Type Checking

```python
type()     # Type of object
isinstance() # Check type
issubclass() # Check class
hasattr()   # Has attribute
getattr()   # Get attribute
setattr()   # Set attribute
delattr()   # Delete attribute
```

## Object

```python
id()       # Object identity
hash()     # Hash value
vars()     # __dict__
dir()      # All attributes
callable() # Is callable
help()     # Help
```
