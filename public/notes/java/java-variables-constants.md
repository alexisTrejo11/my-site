# Variables and Constants

> Variable declaration, constants (final), and scope

## Variable Types

```java
// Local variable
void method() {
    int localVar = 10;
}

// Instance variable (field)
class MyClass {
    int instanceVar;
}

// Static variable
class MyClass {
    static int staticVar;
}
```

## Constants

```java
final double PI = 3.14159;
final String APP_NAME = "MyApp";
```
