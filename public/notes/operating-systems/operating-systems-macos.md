# macOS

**macOS** is Apple’s desktop/laptop operating system family. Internally it descends from **NeXTSTEP** and shares a **Unix-like** userland with a **Mach + BSD** heritage in its **[kernel](/learning/operating-systems-kernel)** (branded **XNU**).

## What to remember as a learner

- **APFS** is the default filesystem for modern Macs: snapshots, cloning, encryption integration — see [Memory Storage Filesystems and Paths](/learning/computer-architecture-memory-storage-filesystems-and-paths).
- **Apple Silicon** Macs use **[SoCs](/learning/computer-architecture-soc-and-gpu)** with **unified memory** between CPU and GPU; performance tuning differs from “discrete GPU + separate VRAM” PCs.
- **Code signing** and **Gatekeeper** shape how software is distributed compared with typical Linux desktops.
